# IPL Data Analysis Dashboard

This project is about analyzing IPL (Indian Professional League) cricket data and showing the results in a nice dashboard.

## What This Project Does

- Looks at IPL match data from different seasons
- - Shows which teams won the most
  - - Finds out which players scored the most runs and took the most wickets
    - - Shows Orange Cap and Purple Cap winners
      - - Built a clean and easy to understand dashboard using Power BI
       
        - ## Data Used
       
        - - Ball by ball data of IPL matches
          - - Match results and details
            - - Player information
              - - Team details
               
                - ## Tools Used
               
                - - Power BI (for making the dashboard)
                  - - Excel (for cleaning and organizing data)
                    - - Data Analysis techniques
                     
                      - ## How It Looks
                     
                      - The dashboard shows all the important IPL stats in one place. You can filter by season, team, or player to see different results.
